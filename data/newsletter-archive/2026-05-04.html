THIS WEEK'S PLAY: Same Leads, Different Results — A Case Study

Three insurance agents bought the same batch of aged leads. Same vendor. Same vertical (final expense). Same state. Same lead age (60-90 days). Same price ($0.50/lead). Each bought 200 leads.

Here's what happened:

Agent A — "The Dialer"

Called every lead once. Left voicemails on about half. Didn't text. Didn't email. Spent two days burning through the list and moved on.

MetricAgent A
Leads worked200
Contact attempts per lead1.0
Contacts made31 (15.5%)
Appointments booked6
Policies closed2
Commission earned$1,400
Cost of leads$100
ROI1,300%
Agent A would tell you "aged leads kind of work." 1,300% ROI isn't bad. But it's a fraction of what was possible.

Agent B — "The System"

Ran the full 7-day multi-channel cadence. Phone, text, email — every day for a week per lead. Used vertical-specific final expense scripts. DNC-scrubbed before the first call.

MetricAgent B
Leads worked200
Contact attempts per lead7.2
Contacts made74 (37%)
Appointments booked29
Policies closed16
Commission earned$12,800
Cost of leads$100
ROI12,700%
Same leads. 8x more revenue. The only difference was the process.

Agent C — "The Operator"

Everything Agent B did, plus: imported leads into GoHighLevel with source tags, set up the 7-stage pipeline, automated the cadence triggers, and moved non-responsive leads into a 90-day email nurture drip after Day 7.

MetricAgent C
Leads worked200
Contact attempts per lead7.2 + nurture
Contacts made74 (37%)
Appointments booked29
Policies closed (Week 1-2)16
Policies closed (nurture, 90 days)4
Total policies20
Commission earned$16,000
Cost of leads$100
ROI15,900%
Agent C didn't close more in the first two weeks than Agent B. The 7-day cadence produced identical results. But the automated nurture drip caught 4 more policies over the following 90 days — leads that Agent B had already written off.

Agent A and Agent C bought the same leads for a year. Agent C made $175,200 more. The difference wasn't the leads. It was never the leads.

What Actually Changed

Let's be precise about what separates these three:

1. Contact attempts. Agent A made 1 attempt per lead. Agents B and C made 7+. That single change doubled the contact rate from 15% to 37%.

2. Multi-channel approach. Agent A called only. B and C used phone, text, and email. The leads who didn't answer the phone responded to texts. The leads who ignored texts engaged with email. Every channel catches leads the other channels miss.

3. Scripts. Agent A winged it. B and C used final expense-specific scripts that opened with empathy and the prospect's original inquiry. Conversations were longer and appointments were more qualified.

4. The long tail. Only Agent C captured the 90-day nurture conversions. Those 4 extra policies per month add up to 48 per year — $38,400 in commission from leads that both A and B had abandoned.

None of these require more skill. Agent A might be the best closer in the room. But closing ability doesn't matter if you only talk to 31 people instead of 74.
