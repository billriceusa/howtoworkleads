## Outline

### H1: GoHighLevel Setup Guide: Build an Aged Lead Campaign That Converts

### H2: Why GoHighLevel Is the Best Platform for Aged Leads
- All-in-one: CRM + dialer + SMS + email + automation + landing pages
- Built-in multi-channel sequences (exactly what aged leads need)
- Affordable: $97-$297/mo covers everything vs. 4-5 separate tools
- Massive agent community (insurance + mortgage heavy)

### H2: Step 1 — Import Your Aged Leads
- CSV formatting requirements (columns: first name, last name, phone, email, address, lead source, lead age)
- How to import: Settings → Contacts → Import
- Tagging on import: add tags for lead source, industry, lead age bracket
- DNC scrubbing reminder: scrub BEFORE importing
- Common import errors and how to fix them

### H2: Step 2 — Build Your Aged Lead Pipeline
- Create a dedicated pipeline (separate from fresh leads)
- Recommended stages:
  1. New / Imported
  2. Contact Attempted
  3. Contacted / Warm
  4. Appointment Set
  5. Proposal / Quote Sent
  6. Closed / Won
  7. Not Interested
  8. Bad Data
- Pipeline settings: auto-move rules, stage duration alerts

### H2: Step 3 — Create Your Multi-Channel Workflow
- Workflow builder walkthrough with specific trigger → action chains
- **Trigger:** Contact added to pipeline stage "New / Imported"
- **Day 1 Actions:**
  - Send SMS (template: intro text)
  - Wait 2 hours
  - Send voicemail drop (if using GHL's VM drop feature)
  - Create task: "Call [contact name]"
- **Day 2 Actions:**
  - Send email (template: value-based first email)
- **Day 3 Actions:**
  - Create task: "Follow-up call"
  - Send SMS (template: check-in text)
- Continue through Day 7 cadence
- **Day 7 Action:** Send breakup email → Move to "Not Interested" if no engagement → Move to long-term nurture workflow
- Include screenshots/descriptions of each workflow step

### H2: Step 4 — Set Up Templates
- **SMS templates** (3 aged lead texts — intro, follow-up, breakup)
- **Email templates** (4 emails — intro, value, social proof, breakup)
- **Voicemail drop recordings** (3 voicemails for the cadence)
- How to create and save templates in GHL
- Link to scripts/templates post for actual script content

### H2: Step 5 — Configure Your Dialer
- GHL built-in dialer setup
- Local presence dialing (display local area code)
- Call recording settings (compliance — check state laws)
- Power dialer for batch calling aged leads
- Connecting to Twilio for SMS/calling

### H2: Step 6 — Build Reporting Dashboards
- Key metrics to track: contact rate, response rate, appointment rate, close rate
- How to create custom reports in GHL
- Pipeline velocity tracking
- Revenue attribution

### H2: Step 7 — Long-Term Nurture Automation
- After the 7-day active cadence, move unresponsive leads to nurture
- Monthly email drip: industry updates, rate changes, seasonal reminders
- Re-engagement trigger: if lead opens email or clicks link → move back to active pipeline
- Birthday/anniversary automation (if data available)

### H2: Pro Tips for GHL Aged Lead Campaigns
- Batch imports: import 200-500 leads at a time, not all 2,000 at once (prevents overwhelm)
- A/B test: create two workflow variants, split leads, compare results
- Compliance: always include opt-out in SMS and email, respect DNC
- Speed: process the pipeline daily — don't let tasks stack up

### H2: FAQ
1. **Can GoHighLevel handle bulk aged lead imports?** [Yes — CSV import with tagging, up to thousands at a time]
2. **How much does GoHighLevel cost?** [$97/mo Starter, $297/mo Unlimited — most agents need Starter]
3. **Do I need Twilio for GHL?** [Yes for SMS and calling — additional cost of ~$0.01-$0.02 per message/minute]
4. **Can I use GHL for both aged and fresh leads?** [Yes — create separate pipelines and workflows for each]
5. **How long does it take to set up?** [2-4 hours for initial setup following this guide, then ongoing optimization]
